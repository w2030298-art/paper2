# Model Change Inventory

- MEC task model changes: scalar tasks become single-node DAG specs; DAG tasks are opt-in.
- Queue model changes: MM1, MMC, parallel, and finite-capacity models are available.
- Heterogeneous cooperative edge changes: BS default remains, RSU/UAV/peer/cloud are opt-in.
- Energy model changes: network-level DVFS energy is split across local, tx, edge, and migration.
- Mobility model changes: Markov mobility and trace adapters are separated.
- Communication model changes: analytic vs 3GPP-lite communication model split.
- Dynamic pricing changes: queue/channel/migration-aware pricing replaces static-only interpretation when enabled.
- Game-aware critic / primal-dual changes: critic features include pricing and constraint residuals.
