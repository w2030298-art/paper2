# Mainline-A Experiment Protocol

## Evidence Levels

- N0: smoke correctness.
- N1: small-scale oracle comparison.
- N2: controlled ablation.
- N3: OOD generalization.

## N1 Oracle

N1 must include a small-scale optimum or enumeration oracle and report oracle gap.

## N2 Ablation

N2 must cover no_price, no_queue, no_migration, no_dual, and no_cooperation.

## N3 OOD

OOD tests must cover user count, edge count, mobility intensity, and channel model shifts.
